# Shopping MasterMind

Shopping MasterMind is a Flutter application designed for efficient shopping management.

## Screens

- **Welcome Screen**
- **Signup Screen**
- **Login Screen**
- **Add Item Screen**
- **Calendar Screen**
- **Categories Screen**
- **Dashboard Screen**
- **FAQ Screen**
- **Items List Screen**
- **Settings Screen**

## Getting Started

To run this Flutter project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Navigate to the project directory:

   ```bash
   cd your-repository
   ```

3. Run the project:

   ```bash
   flutter run
   ```
